# Smart Auto Web Scraper

Automatically extracts data from ANY website.
Zero CSS selectors. Zero manual inspection. Just paste a URL!

##  Features
- Auto detects ALL data patterns on any page
- Shows real sample data for every field
- Pick specific fields by number only
- Keyword filtering built in
- Saves as CSV, JSON and TXT
- Email alerts for new listings

##  Tech Stack
- Python 3
- BeautifulSoup4
- Requests
- smtplib
- python-dotenv

##  Setup

### 1. Install libraries
pip install requests beautifulsoup4 python-dotenv

### 2. Create .env file
EMAIL_SENDER=youremail@gmail.com
EMAIL_PASSWORD=your_app_password
EMAIL_RECEIVER=youremail@gmail.com

### 3. Run
python main.py

##  What It Can Scrape
- Job listings
- News articles
- Product prices
- Quotes
- Tables
- Any repeated data pattern
